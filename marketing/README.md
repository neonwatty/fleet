# Fleet marketing drafts

This directory holds reusable launch and promotional copy for Fleet.

- `launch-post.md` - longer launch announcement.
- `social-copy.md` - short post and thread variants.
- `community-posts.md` - Hacker News and Reddit variants.
- `demo-script.md` - screen recording outline.
- `launch-checklist.md` - pre-merge and pre-promotion checklist.
- `press-kit.md` - concise product facts for reuse.
- `screenshot-guide.md` - capture plan for product screenshots and demo media.

Keep claims aligned with the product as implemented in the root `README.md` and
`menubar/README.md`. Fleet's sharp positioning is local Mac capacity management
for coding-agent work, not general-purpose cluster orchestration.
