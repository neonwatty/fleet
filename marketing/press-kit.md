# Press kit

## Product

Fleet is a local Mac fleet control plane for coding-agent sessions.

## One-liner

Run more Claude Code and Codex-style sessions across your Macs without manually juggling SSH, memory pressure, localhost tunnels, or cleanup.

## Core features

- Automatic machine placement based on online status, memory headroom, and swap pressure.
- Fresh git worktrees for each launched session.
- SSH tunnels that keep remote dev servers reachable from localhost.
- Pinned tunnel ports for OAuth callbacks.
- Live Bubble Tea terminal dashboard.
- Native macOS menu bar app.
- Agent-readable JSON via `fleet status --json`.
- Cleanup and diagnostics with `fleet clean` and `fleet doctor`.

## Audience

Developers using a MacBook plus nearby Mac minis or other Macs as local capacity for multiple coding-agent sessions.

## Positioning

Fleet is not a general-purpose orchestrator. It is a narrow, transparent tool for local Mac development machines connected by SSH.
